find_utf8bom
==================

소스코드중 의도치않게 UTF8 With BOM 으로 저장되는 일이 생겨서 만들어봄.

install :
-------------

* install golang latest. (https://golang.org/dl/)
* download src (git clone https://github.com/jaksal/find_utf8bom.git)
* cd find_utf8bom 
* go build

usage :
-------------

```
find_utf8bom [path]
find_utf8bom ./src
```
