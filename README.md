find_utf8bom
==================

find utf8 with BOM encoding file

install :
-------------

* install golang latest. (https://golang.org/dl/)
* download src (git clone https://github.com/jaksal/find_utf8bom.git)
* cd find_utf8bom 
* go get && go build

usage :
-------------

```
find_utf8bom path=~/src
```
